## 🎯 Sobre o Projeto
Este é um projeto de aplicação web **end-to-end** desenvolvido para monitoramento e conversão de moedas em tempo real. Como estudante de **Ciência de Dados**, foquei na integração entre o processamento de dados dinâmicos e uma interface de usuário intuitiva.

O sistema consome dados da **AwesomeAPI**, realiza o tratamento de JSON no Back-end e entrega os resultados renderizados via templates dinâmicos.

## 🚀 Tecnologias Utilizadas
- **Back-end:** Python 3.12 + Flask
- **Integração:** Requests (Consumo de APIs REST)
- **Front-end:** HTML5, CSS3 (Design Responsivo) e Jinja2
- **Ambiente:** Gestão de dependências via PIP
